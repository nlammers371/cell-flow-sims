# cell-flow-sims

Phase 1 implementation of a cell dynamics simulator on a sphere.

## Quick start

```bash
python -m venv .venv
source .venv/bin/activate
pip install -e .[test]
pytest
python examples/run_minimal.py
```

## Napari UI (Phase 2)

```bash
napari
```

In napari: Plugins -> Cell Sphere Sim.
